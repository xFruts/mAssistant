Данный сервис это телеграмм бот, который помогает пользователям в чат группах.
Его функционал включает в себя:
- Автоматическое приветствие новых участников.
- Модерация сообщений (удаление спама, нецензурной лексики).
- Ежедневное сообщение с погодой в заданном городе.
- Напоминания о важных событиях/задачах на сегодня/завтра.


Для запуска бота необходимо:
- Склонировать репозиторий.
```bash
git clone https://github.com/xFruts/mAssistant
```
- Создать файл `.env` в корне проекта и заполнить его следующими переменными:
```
BOT_TOKEN=ваш_токен_бота_из_telegram
WEATHER_API_KEY=ваш_ключ_от_сервиса_погоды
DB_USER=имя_пользователя_базы_данных
DB_PASSWORD=пароль_пользователя_базы_данных
```
- Установить Docker и Docker Compose.
- Собрать Docker образ.
```bash
docker-compose up -d
```
